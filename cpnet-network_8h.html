CPNET_NETWORK_API int net_init ( )

Initialize networking interface.

Calling this function on POSIX platforms (or wherever BSD sockets are supported) has no effect. On Windows platforms, this function initializes Winsock API by calling WSAStartup.

CPNET_NETWORK_API int net_setopt ( socket_t  s,
int  option 
)

Sets option on socket. Only options that could be treated as boolean can be set by this function. For other usages see net_setval .

Supported options are:

  • SO_BROADCAST
  • SO_DEBUG
  • SO_DONTROUTE
  • SO_KEEPALIVE
  • SO_OOBINLINE
  • SO_REUSEADDR

NOTE: calling this function with other types of options, will cause a logical error and will behave differently on different platforms.
